# ملاحظات حول صفحة دليل الحسابات الاحترافية

## الحالة الحالية
تم إنشاء ملف `AccountsPagePro.tsx` الذي يحتوي على واجهة احترافية جديدة لدليل الحسابات.

## المكونات المستخدمة من Shadcn/ui
- Card, CardContent, CardDescription, CardHeader, CardTitle
- Button
- Input
- Label
- Select, SelectContent, SelectItem, SelectTrigger, SelectValue
- Dialog, DialogContent, DialogDescription, DialogHeader, DialogTitle
- Tabs, TabsContent, TabsList, TabsTrigger
- Badge
- Switch
- Textarea
- AlertDialog للتأكيد قبل الحذف

## الميزات الرئيسية
1. **تصميم احترافي**: تدرجات لونية متناسقة مع النظام
2. **تبويبات ذكية**: تصنيف الحسابات حسب النوع (أصول، التزامات، حقوق ملكية، إيرادات، مصروفات)
3. **بحث متقدم**: إمكانية البحث في الحسابات
4. **نموذج إضافة/تعديل متكامل**: يشمل جميع الحقول المطلوبة
5. **دعم RTL**: تصميم متوافق مع اللغة العربية
